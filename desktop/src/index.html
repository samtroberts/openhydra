OpenHydra
Decentralised P2P LLM Inference Network

OpenHydra is an open-source peer-to-peer network that lets anyone contribute compute to run large language model inference collaboratively — no cloud required. Each node earns HYDRA tokens for serving requests.

This desktop application manages your local OpenHydra node. It starts the DHT bootstrap service and the inference node as background processes, and provides a chat interface so you can test the network directly from your machine.
